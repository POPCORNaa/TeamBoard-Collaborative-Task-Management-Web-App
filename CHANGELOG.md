# Changelog

## [v0.1.0] - 2025-10-22

### Added
- Initial public release of the project.
- Frontend and backend separated architecture.
- Basic API endpoints implemented (e.g., user authentication, data retrieval).

### Changed
- Updated project structure to support independent frontend and backend builds.
- Improved environment variable loading logic.

### Fixed
- Fixed backend environment variable path issues.
- Fixed frontend build resource path problems.

### Removed
- Removed old test scripts and unused configuration files.
